¿Qué es todo esto?

Init


          git init
  • Inicializa el repositorio
  • Crea una carpeta .git

Touch


          touch fichero.txt
  • Crea un fichero (unstaged)

Add


          git add fichero.txt
  • Controla el estado del fichero
  • Añade los cambios para hacer commit (staged)

Status


          git status
  • Muestra el estado del repositorio
    • Ficheros añadidos
    • Ficheros borrados
    • Ficheros movidos

Commit


          git commit -m "Initial commit!"
  • Realiza una instantánea de los ficheros
  • Incluye un mensaje

Crear una cuenta en GitHub

  1. Entrar en github.com y rellenamos los datos.
  2. Elegimos plan Free.
  3. Configuramos nuestro nombre de usuario: 
    git config --global user.name usuario
  4. Configuramos nuestro correo: 
    git config --global user.email usuario@gmail.com

Crear un repositorio en GitHub (I)

En la barra de tareas superior seleccionamos el símbolo '+' para añadir nuevo repositorio.

Crear un repositorio en GitHub (II)

Le damos un nombre y una descripción a nuestro proyecto.

Crear un repositorio en GitHub (III)

Elegimos crear un repositorio público y creamos respositorio.

Añadimos el respositorio remoto

Para añadir el repositorio como remoto de nuestro repositorio local usaremos:

git remote add origin https://github.com/usuario/my-first-repo

Subimos los cambios a nuestro repositorio remoto

Para sincronizar nuestro repositorio local con el repositorio remoto que acabamos de crear usamos:

git push origin master
